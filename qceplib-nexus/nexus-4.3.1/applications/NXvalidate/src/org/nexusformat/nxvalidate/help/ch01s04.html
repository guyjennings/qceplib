Downloading the NXvalidate JAR File

The most convenient form for downloading is to pick up a single installer Jar file containing the required classes:

  1. NXvalidate-v1.0alpha.jar (MB) - NXvalidate application installer (no NXconvert)
  2. NXvalidate-v1.0alpha-win32.jar (MB) - NXvalidate application installer with Windows 32bit NXconvert.
  3. NXvalidate-v1.0alpha-win64.jar (MB) - NXvalidate application installer with Windows 64bit NXconvert.
  4. NXvalidate-v1.0alpha-lnx64.jar (MB) - NXvalidate application installer with Linux 64bit NXconvert.
  5. NXvalidate-v1.0alpha-lnx32.jar (MB) - NXvalidate application installer with Linux 32bit NXconvert.
  6. NXvalidate-doc-v1.0alpha.zip (MB) - Javadocs API documentation.
  7. NXvalidate-Tutorial.zip (MB) - this tutorial (also included as help in the application.)
  8. NXvalidate-all-v1.0alpha-source.zip (MB) - the source code for NXvalidate.

(Note: if you try to download these directly your browser may say something about a failed security check. Make sure that you save it to a file, for instance by using shift-click in Netscape).
