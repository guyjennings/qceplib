Example

An example can be downloaded here:

  1. example.zip (MB) - Example (used in this tutorial).

This example can be opened with NXvalidate.

To use the example unzip the zip file which should give you two files:

  1. NXmonopd.hdf - the data file.
  2. NXmonopd.nxdl.xml - the NXDL definition file.

These two files can be opened with the NXvalidate GUI.
