NXvalidate Main Features

  1. Opens NeXus files in HDF4/5 or XML format.
  2. Validates a single NeXus file against a user selected NXDL description.
  3. Allows batch validation of a selection of NeXus files.
  4. File structure displayed as a tree.
  5. Metadata content of the file is displayed.
  6. Results of the validation and the errors are displayed.
  7. Results of the validation can be saved.
  8. Integrated help and tutorial.
